# 🩺 Diabetes Prediction Machine Learning Dashboard

A **beautiful interactive Machine Learning dashboard** built using **Streamlit, Python, and Scikit-Learn** to analyze diabetes data and train predictive models.

This project demonstrates **data preprocessing, visualization, machine learning training, and model evaluation** in a clean and interactive dashboard interface.

# 📊 Project Overview

The goal of this project is to build a **Machine Learning dashboard** that:

* Analyzes diabetes dataset features
* Performs **data preprocessing**
* Visualizes important patterns
* Trains **multiple ML models**
* Compares model performance

The dashboard allows users to **explore the dataset and train models interactively**.

---

# 🧠 Machine Learning Models Used

The following models are implemented:

* Logistic Regression
* Random Forest Classifier

The models are trained after preprocessing and evaluated using **accuracy metrics**.

---

# 📂 Dataset Information

Dataset used: **Diabetes Prediction Dataset**

Features include:

* Gender
* Age
* Hypertension
* Heart Disease
* Smoking History
* BMI
* HbA1c Level
* Blood Glucose Level

Target variable:

* **Diabetes (0 = No, 1 = Yes)**

---

# ⚙️ Data Preprocessing

The following preprocessing techniques were applied:

* Label Encoding for categorical variables
* Feature scaling using **StandardScaler**
* Train-test split (80% training / 20% testing)
* Feature-target separation

This ensures the data is **properly prepared for machine learning models**.
